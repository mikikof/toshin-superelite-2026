## このリポジトリ

講義中に投影し、生徒に開放する **4 つの web デッキ + ハブサイト** を配布するための静的サイト。
GitHub Pages で公開され、URL を知っている人なら誰でも閲覧できます(認証なし)。
HTML/CSS/JS のみで動作し、サーバー側処理はありません。

## 構成

```
toshin-superelite-2026/
├── index.html        ← ハブ(4幕への入口)
├── act0/index.html   ← 第0幕 開幕(10分)
├── act1/index.html   ← 第1幕 データの不均衡(30分)
├── act3a/index.html  ← 第3幕a Transformer(25分)
├── act3b/index.html  ← 第3幕b 拡散モデル(25分)
├── _tools/           ← act0 が参照する画像資産(顔100枚・X線等)
└── _assets/          ← ATLAS Higgs 画像
```

## 操作

| キー | 動作 |
|---|---|
| `→` `Space` | 次へ |
| `←` | 戻る |
| `0`–`9` | スライド番号ジャンプ |
| `J` | 目次 |
| `H` | 進行役(みき)の吹き出し ON/OFF |
| `P` | 本文ハイライト走査(文字ポップ) |
| `Esc` | 補足モーダルを閉じる |

各カードの「補足」チップをクリックすると、難所の解説モーダルが開きます。

## 動作環境

- Chrome / Edge / Safari の最新版(2024 以降)
- PC・タブレット・スマホ全対応(レスポンシブ)
- インターネット回線必須(Google Fonts を CDN から読み込みます)
