このツールは何を見せている？

原点 (0,0) でいちばん低くなる 損失の地形（細長い谷）を、 ロボット θ=(x,y) が下っていきます。右の図は x–y 平面を真上から見たもので、 明るいほど低く、赤い目印がゴールです。

損失（谷の形）
L(x,y)= 12( u2a2+ v2b2)
L = ½(u²/a² + v²/b²)
座標の回転（谷の向き φ）
[u,v]T= R(φ)[x,y]T
[u,v]ᵀ = R(φ)[x,y]ᵀ
  • θ=(x,y)：ロボの位置。最小化したいパラメータ。
  • a, b：谷の細さ。b≫a で横長の谷になる。
  • φ：谷の向き（回転角）。
SGD  : θₜ₊₁ = θₜ − η·∇L(θₜ) Adam : mₜ = β₁mₜ₋₁ + (1−β₁)gₜ ,  vₜ = β₂vₜ₋₁ + (1−β₂)gₜ²        θₜ₊₁ = θₜ − η·m̂ₜ /(√v̂ₜ + ε) AdamW: θₜ₊₁ = θₜ − η·( m̂ₜ/(√v̂ₜ+ε) + λθₜ )
